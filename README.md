# 🏝️ Nova Nest Retreats

A premium resort booking and management web application.

## Project Structure

```
NOVA-NEST-RETREATS/
│
├── index.html                  # Landing / Home page
│
├── pages/                      # All HTML pages
│   ├── resorts.html            # Resort listings
│   ├── resort-details.html     # Individual resort details
│   ├── packages.html           # Travel packages
│   ├── offers.html             # Special offers & deals
│   ├── about.html              # About us
│   ├── contact.html            # Contact form
│   ├── faq.html                # Frequently asked questions
│   ├── privacy-policy.html     # Privacy policy
│   ├── login.html              # User login
│   ├── signup.html             # User registration
│   ├── user-dashboard.html     # User dashboard
│   ├── admin-dashboard.html    # Admin dashboard
│   ├── bookings.html           # User bookings
│   ├── wishlist.html           # User wishlist
│   ├── payment-history.html    # Payment history
│   ├── settings.html           # User settings
│   ├── admin-bookings.html     # Admin booking management
│   ├── admin-resorts.html      # Admin resort management
│   ├── admin-users.html        # Admin user management
│   ├── sitemap.html            # Sitemap
│   └── 404.html                # 404 error page
│
├── css/                        # Stylesheets
│   ├── style.css               # Main styles
│   ├── responsive.css          # Responsive / media queries
│   └── animations.css          # CSS animations & transitions
│
├── js/                         # JavaScript files
│   ├── main.js                 # Core app logic
│   ├── auth.js                 # Authentication logic
│   ├── dashboard.js            # Dashboard functionality
│   ├── booking.js              # Booking system logic
│   ├── animations.js           # Animation controllers
│   ├── filters.js              # Search & filter logic
│   ├── slider.js               # Image slider / carousel
│   └── countdown.js            # Countdown timer for offers
│
├── assets/                     # Static assets
│   ├── images/
│   │   ├── resorts/            # Resort images
│   │   ├── users/              # User avatars
│   │   ├── banners/            # Banner images
│   │   └── backgrounds/        # Background images
│   ├── icons/                  # Icon assets
│   ├── vectors/                # SVG / vector graphics
│   ├── videos/                 # Video assets
│   └── logos/                  # Brand logos
│
└── README.md                   # Project documentation
```

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Vanilla CSS with animations
- **JavaScript** — Vanilla JS, no frameworks

## Getting Started

Open `index.html` in your browser to view the application.
